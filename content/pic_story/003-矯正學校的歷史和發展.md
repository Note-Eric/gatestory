## 台灣少年矯正教育的歷史與發展

台灣少年矯正教育的歷史與發展分為以下幾個階段：

第一階段：日治時期。當時，已有專門關押少年犯人的矯正機構，並提供矯正及教育訓練，以控制他們的行為問題。這個階段的少年矯正主要著重於青少年的道德教化和紀律體能訓練為主。

第二階段：戰後初期。西元1945年中華民國政府接管台灣後，少年矯正重點是以紀律管理和職業訓練為主，以期將少年犯人納入社會，並提高其生活自理和就業能力。這個階段的這些機構對少年犯人實行嚴格的紀律管理，例如限制行動自由、禁止通訊和家屬探視等，以期震懾犯罪心理，增強紀律性和服從性。此外，職業訓練也是重點之一。矯正機構為少年犯人提供各種職業技能訓練，例如木工、汽車修護和美髮等，以提高其生活自理和就業能力。

第三階段：現代化的轉型期。隨著社會變遷和法制環境的改變，少年矯正機構的功能逐漸轉型，強調犯罪人的品德教育和社會適應訓練，並且注重社會保護和照顧特殊需求的人群。此外，也開始推動犯罪預防教育，以減少未成年人犯罪發生的可能性。

第四階段：全人發展的時期。西元1999年成立明陽中學與誠正中學2所少年矯正學校後，矯正學校不僅承擔少年繼續接受中等教育學程和矯正不良習性的功能，也陸續注意犯罪少年關於妨害性自主、毒品、身心障礙、家庭暴力等特殊犯罪議題的個別化治療與矯正，目標是協助犯罪少年重新適能社會生活。

隨著2021年8月1日勵志中學與敦品中學二所少年輔育院，正式改制為矯正學校，矯正學校亦不再是單純的犯罪少年收容及教育功能，而是成為一個全面性和多元性機構，提供犯罪少年在學習、職業訓練、心理治療、特殊教育、社會適應訓練、家庭參與及矯正後支持等多方面的協助。

由歷史的脈絡來縱觀矯正教育的發展，現代的矯正學校對收容學生的處遇，已逐步朝向個別化與多元化，亦被期許更為專業化的犯罪少年矯正模式。

## 矯正學校的歷史沿革
### 明陽中學
| 年份                    | 事件描述                                                                                     |
|-------------------------|----------------------------------------------------------------------------------------------|
| 民國45年（西元1956年） | 隸屬臺灣省政府社會處，成立「臺灣省立高雄少年感化院」，原址高雄市苓雅區                           |
| 民國48年（西元1959年） | 更名為「臺灣省立高雄少年輔育院」                                                             |
| 民國70年（西元1981年） | 改隸法務部，復更名為「臺灣高雄少年輔育院」，收容臺灣省南部7縣市暨福建省金門縣等所在地之地方法院少年法庭裁定交付感化教育之男性少年受處分人 |
| 民國84年（西元1985年） | 遷至高雄縣燕巢鄉現址                                                                         |
| 民國88年（西元1999年） | 「臺灣高雄少年輔育院」改置為矯正學校「明陽中學」，第一批收容學生為臺灣新竹少年監獄之少年受刑人，至今（民國113年）收容對象仍為少年徒刑、拘役之少年受刑人 |
| 民國100年（西元2011年）| 改隸屬法務部矯正署                                                                          |
| 民國101年（西元2012年）| 增設高雄少年觀護所燕巢分所及其附設觀察勒戒處所，收容未成年女性受刑人及收容少女及勒戒少女          |

### 誠正中學
| 年份                    | 事件描述                                                                                     |
|-------------------------|----------------------------------------------------------------------------------------------|
| 民國88年（西元1999年） | 7月1日成立，為全國第一所以感化教育受處分人為收容對象之少年矯正學校，校址於新竹縣新豐鄉，校名「誠正中學」 |
| 民國100年（西元2011年）| 改隸屬法務部矯正署                                                                          |

### 敦品中學
| 年份                    | 事件描述                                                                                     |
|-------------------------|----------------------------------------------------------------------------------------------|
| 民國45年（西元1956年） | 隸屬臺灣省政府社會處，成立「臺灣省立臨時少年感化院」，臨時院址創始於臺灣新竹少年監獄內，當時收容少年超過800人。曾更名為「臺灣省立新竹少年感化院」、「臺灣省立台北少年感化院」 |
| 民國47年（西元1958年） | 遷址至桃園鄉林投店甲種國民兵訓練營房為院址，復更名為「臺灣省立桃園少年感化院」                   |
| 民國48年（西元1959年） | 更名為「臺灣省立桃園少年輔育院」                                                             |
| 民國70年（西元1981年） | 改隸法務部，復更名為「臺灣桃園少年輔育院」                                                   |
| 民國100年（西元2011年）| 改隸屬法務部矯正署                                                                          |
| 民國108年（西元2019年）| 依「少年輔育院改制矯正學校計畫」，因應改制矯正學校成立之準備，設置「誠正中學桃園分校」             |
| 民國110年（西元2021年）| 「誠正中學桃園分校」正式改制成矯正學校，校名為「敦品中學」                                      |
